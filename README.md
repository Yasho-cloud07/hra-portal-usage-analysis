Data-driven analysis of HRA Portal usage using Amazon CloudFront logs, focusing on RUI and EUI interactions, user behavior trends, and growth patterns, presented through an interactive report for stakeholders.
# HRA Portal — RUI & EUI Product Usage Analysis

## Overview

This project analyzes product usage patterns in the HRA Portal using Amazon CloudFront logs.
It focuses on two major components:

* **RUI (Registration User Interface)**
* **EUI (Exploration User Interface)**

The analysis spans ~28 months of data and ~6.8 million events.

---

## Project Files

* `/report/index.html` → Final interactive report/dashboard
* `/data/` → Dataset (CSV files)
* `/scripts/` → Data processing scripts 
* `/outputs/` → Generated charts and results
